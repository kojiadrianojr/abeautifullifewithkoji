# Throwback Photos

Place your throwback/nostalgic photos here.

## Instructions

1. Add your throwback photos to this directory
2. Name them descriptively (e.g., `photo1.jpg`, `photo2.jpg`)
3. Update the `throwbackPhotos` array in `/config/wedding.json` with the file paths

## Recommended

- **Format**: JPG or PNG
- **Dimensions**: Square format (1:1 ratio) works best (e.g., 1000x1000px)
- **File Size**: Under 2MB per photo for optimal loading
- **Number**: 4-8 photos work best for the grid layout

## Example Configuration

```json
"throwbackPhotos": [
  "/images/throwback/photo1.jpg",
  "/images/throwback/photo2.jpg",
  "/images/throwback/photo3.jpg",
  "/images/throwback/photo4.jpg"
]
```
