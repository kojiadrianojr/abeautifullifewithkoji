# Hero Album Images

This folder contains the images that will be **automatically displayed** in the clickable hero image carousel on your wedding website.

## Adding Images

1. Simply add your hero images to this folder with any filename you prefer
2. Supported formats: JPG, JPEG, PNG, WebP, GIF, AVIF
3. Recommended dimensions: 600x800px or similar portrait orientation
4. File naming: Use any naming scheme you like - images will be sorted naturally

## Automatic Detection

- **No configuration needed!** All images in this folder are automatically detected
- Images are sorted naturally (1.jpg, 2.jpg, 10.jpg, or alphabetically)
- No minimum or maximum number of images
- If no images are found, the hero image component will be hidden

## Features

- Single image: Displays without navigation controls
- Multiple images: Click to cycle through, or use arrow buttons
- Navigation dots show current image and total count
- Smooth fade animations between images
- Fully responsive and accessible

## Example Structure

```
hero-album/
├── 1.jpg              (main couple photo)
├── 2.jpg              (engagement photo)
├── 3.jpg              (candid moment)
├── 4.jpg              (outdoor shot)
├── proposal.jpg        (works with any name)
└── beach-sunset.jpg    (descriptive names OK too)
```

**Pro Tip:** Start filenames with numbers (1-, 2-, 3-) for precise ordering, or use descriptive names for easier management.
