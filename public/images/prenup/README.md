# Prenup Photos

Place your prenuptial/engagement photos here.

## Instructions

1. Add your prenup photos to this directory
2. Name them descriptively (e.g., `photo1.jpg`, `photo2.jpg`)
3. Update the `prenupPhotos` array in `/config/wedding.json` with the file paths

## Recommended

- **Format**: JPG or PNG
- **Dimensions**: Square format (1:1 ratio) works best (e.g., 1000x1000px)
- **File Size**: Under 2MB per photo for optimal loading
- **Number**: 4-8 photos work best for the grid layout

## Example Configuration

```json
"prenupPhotos": [
  "/images/prenup/photo1.jpg",
  "/images/prenup/photo2.jpg",
  "/images/prenup/photo3.jpg",
  "/images/prenup/photo4.jpg"
]
```
