# Selling This Template as a Product

This guide is for developers who want to sell this wedding website template as a product.

### 1. One-Time Purchase
- Sell the template for $50-$200
- Provide the code + documentation
- Customer customizes themselves

### 2. Customization Service
- Template: $100

- You customize it for them

- Template + Hosting: $30/month

- Easiest for customers

## Customer Support

### What to Offer

**Basic Support**:
- Email support
- 48-hour response time
- Technical questions only

**Premium Support**:
- Video calls for setup
- Screen sharing assistance
- Same-day response
- Deployment help

### Common Support Questions

1. "How do I change colors?"
   - Point to CUSTOMIZATION.md section 3

2. "Images aren't showing"
   - Check image paths
   - Verify files in public/images

3. "How do I deploy?"
   - Send DEPLOYMENT.md
   - Recommend Vercel for ease

4. "Can you add a feature?"
   - Custom work (charge extra)
   - Or add to template roadmap
