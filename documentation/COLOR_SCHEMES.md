# Example Color Schemes

Copy any of these color schemes into your `config/wedding.json` file.

## Romantic Blush

Soft, romantic pink and cream tones perfect for spring/summer weddings.

```json
{
  "theme": {
    "colors": {
      "primary": "#C9A0A0",
      "secondary": "#F4E1D2",
      "accent": "#F8EDE3",
      "background": "#FFFBF7",
      "foreground": "#3D3D3D"
    }
  }
}
```

## How to Use

1. Choose your favorite color scheme
2. Open `config/wedding.json`
3. Find the `"theme"` section
4. Replace the colors with your chosen scheme
5. Save and refresh your browser

## Creating Custom Colors

Use these tools to create your own color scheme:

- [Coolors.co](https://coolors.co) - Generate color palettes
- [Adobe Color](https://color.adobe.com) - Create harmonious colors
- [Canva Colors](https://www.canva.com/colors/) - Wedding color palettes

### Tips for Choosing Colors

1. **Start with your wedding colors** - Match your invitations or theme
2. **Consider your venue** - Coordinate with the space
3. **Think about season** - Lighter colors for spring/summer, richer for fall/winter
4. **Test readability** - Make sure text is easy to read
5. **Preview on mobile** - Check how colors look on different screens
